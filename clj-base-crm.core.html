The spec of the Base CRM API which is currently supported by this library.

create

(create resource data)(create resource resource-ids data)

Create a new resource record.

You should use the arity with resource-ids when the resource you are creating has an association to another resource. (e.g. Associated Contacts)

##resource-ids

{:deal_id 123} - Endpoints with associations to other resources. (e.g. Associated Contacts)

The resource-ids map to the Base API Endpoints

upsert

(upsert resource {:keys [data filters]})

Create a new resource record or update an existing resource record, based on a value of a filter or a set of filters. At least one filter is required.

##Behaviour

  • If multiple records match a set of filters, the request will return an error - 409.
  • If a single record matches, then the existing record is updated
  • If no record matches the query, a new record is created.

##filters

{:email "john.doe@fakemail.com"} - Filters by email {:custom_fields {:external_id 123}} - Filters by the custom field ‘external_id’
